INSTALL
Upload the contents of the upload folder to the root directory containing Xenforo.  
Install the xml file from the control panel.
Be sure to enable view permissions for each usergroup you wish to allow privileges.
Users who have read thread is based on the Xenforo setting for Read Marking Data Lifetime (Days).  Data past this length is deleted and you will lose those reads.  Increase as desired.
